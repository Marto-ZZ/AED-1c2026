## Sobre las prácticas

En las carpetas de tipo `practica-i` lo mas importante es el archivo `practica-i-sol.pdf`, que es la práctica actualizada junto con ejercicios resueltos.
Importantísimo entender que las guías estan sujetas a tener errores propios, no tomen mis respuestas como verdad absoluta, por eso es recomendable que si llegas a encontrar algún ejercicio donde yo la pifié mandes un pull request o escribas un issue explicándolo

Cada pdf se puede volver un toque insoportable bajar 30+ páginas para ir a cada ejercicio, navega entre ejercicios con los botones dentro del índice y el "volver al índice" en los pie de página.

# Más en detalle sobre este repo

Bueno este repo sale totalmente inspirado del repo de Álgebra I de [@naD GarRaz](https://www.github.com/nad-garraz) asi que su dinámica va a ser muy similar. La idea es que este repo (al igual que el de Álgebra I) sea un apunte _dinámico_:

¡Esto está para que _cualquier persona con ganas de contribuir pueda hacerlo_!

## Contribuir

Para contribuir:

**1** - Cloná el repo:
```bash
git clone https://github.com/Marto-ZZ/AED-1c2026
```

**2** - Creá una rama con nombre descriptivo (ej: `ej-5-practica-2` o `taller-3-java`) y pusheala para que los demás se enteren:
```bash
git switch -c nombre-nueva-rama
git push origin nombre-nueva-rama
```

**3** - Trabajá en el archivo del ejercicio que te corresponde. Agregalo y commiteá con un mensaje claro:
```bash
git add ej-5-2.tex
git commit -m "Agrego ej-5-2.tex incisos a, b y c"
```

**4** - Cuando termines, hacé pull por si alguien cambió algo mientras tanto, y luego push:
```bash
git pull
git push origin nombre-nueva-rama
```

**5** - Los cambios van a aparecer en GitHub **en tu rama**. Para fusionar, abrí una **Pull Request** desde la página y esperá el visto bueno para que se mergee. La rama desaparece una vez mergeada.

**6** - Si en tu repo local querés borrar la rama:
```bash
git branch
git switch main
git branch -D nombre-nueva-rama
```

> No hace falta saber LaTeX para contribuir. Si encontrás un error en algún ejercicio, podés abrirlo como un Issue.

Podés usar el código como quieras siguiendo esta licencia: [CC BY-NC](https://creativecommons.org/licenses/by-nc/4.0/)
